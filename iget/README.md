# 项目整体说明

## 项目涉及技术
- 前端框架: vue-cli 2.9.6
- UI框架: element-ui
- 路由管理: vue-rooter
- 滚动插件: better-scroll
- 数据调用: vue-axios
- 数据管理: vuex
- Js语法: Ts
- Css预处理: Stylus
- 包管理工具: yarn

## 项目相关指令
- 项目初始化: vue init webpack iget
- 安装stylus: yarn add stylus stylus-loader -D
- 本地运行项目: npm run dev
- 项目打包上线: npm run build
- 安装element-ui: yarn add element-ui
- 安装better-scroll: yarn add better-scroll
